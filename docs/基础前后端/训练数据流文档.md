## 1. 数据流总览

| 阶段 | 前端操作 | 前端文件与函数 | 前后端通信 | 后端文件与函数 | 数据处理与存储 |

| 2. 训练参数配置 | 用户填写训练参数并点击"开始训练" | events.js: handleStartTraining() | POST /start_training (FormData) | training_service.py: start_training_task() | 初始化训练状态，启动后台训练线程 |

## 3. 关键数据结构

### 训练状态 (training_state)
```python
training_state = {
    "model_name": None,       # 正在训练的新模型名字
    "status": "idle",         # 'idle', 'training', 'success', 'error'
    "progress": 0,            # 当前 Epoch
    "total": 0,               # 总 Epochs
    "metrics": {},            # 实时指标 (如 box_loss, mAP50)
    "eta": "计算中...",       # 预计剩余时间
    "error_msg": "",          # 错误信息
    "start_time": 0,          # 训练开始时间戳
    "last_epoch_time": 0      # 上一轮结束时间戳
}
```

### 训练记录 (TrainingHistory)
```python
class TrainingHistory(Base):
    __tablename__ = "training_history"
    id = Column(Integer, primary_key=True, index=True, autoincrement=True)
    model_name = Column(String, unique=True, nullable=False, index=True) # 训练产生的文件名 (也是key)
    base_model = Column(String)                                          # 基础模型名
    dataset = Column(String)                                             # 数据集名称
    parameters = Column(String)                                          # 训练参数 (JSON 字符串)
    description = Column(String, default="")                              # 模型介绍
    created_at = Column(DateTime, default=datetime.now)                  # 训练完成时间
```

## 4. 关键API端点

| 端点 | 方法 | 功能 | 参数 | 返回值 |
|:-----|:-----|:-----|:-----|:-------|
| /upload_dataset | POST | 上传并解压数据集 | file (FormData) | {"status": "success", "dataset_path": "..."} |
| /start_training | POST | 开始模型训练 | model_name, base_model, dataset_yaml_path, parameters, description (FormData) | {"status": "success", "message": "..."} |
| /training_progress | GET | 获取训练进度 | 无 | training_state字典 |
| /training_history/{model_name} | GET | 获取训练历史 | model_name (路径参数) | {"status": "success", "data": {...}} |
| /update_model_description | POST | 更新模型介绍 | model_name, description (JSON) | {"status": "success"} |

## 5. 数据存储位置

| 数据类型 | 存储位置 | 说明 |
|:---------|:---------|:-----|
| 数据集 | datasets/{dataset_name}/ | 解压后的训练数据集 |
| 基础模型 | models/raw/, models/yolo/ | 原始和内置模型 |
| 训练模型 | models/trained/{model_name}.pt | 用户训练的模型 |
| 训练图表 | backend/trainchart/{model_name}/ | 训练产生的图表 |
| 训练记录 | database/yolo_app.db (TrainingHistory表) | 训练历史记录 |
| 临时训练输出 | runs/detect/{model_name}/ | 训练过程中的临时文件 |
