## 本人毕设项目

### 基于视觉感知和深度学习的产品缺陷检测

#### 项目概览

本项目是一个基于YOLOv11、FastAPI等技术的视觉检测系统，旨在实现产品缺陷的自动检测。系统包括前端和后端两部分，前端使用原生JS和HTML5开发，后端使用FastAPI框架和SQLite数据库。

#### 功能描述

**推理模块：**
系统支持用户上传图片，通过YOLO模型（系统内置了三个yolo模型，也可以使用导入模型和训练模型）进行缺陷检测，并返回检测结果（历史记录可查看）。除此以外，系统还支持用户上传视频和，进行实时检测。
**训练模块：**
系统支持用户上传数据集，通过YOLO模型进行训练，返回训练结果并保存模型。训练的时候可以自定义30种训练参数，如学习率、训练轮数等，还可以查看训练进度，监控参数等（如召回率，准确率），并且系统支持用户组成训练队列，进行多任务训练。
**管理模块：**
系统支持用户管理训练模型、查看训练日志、查看训练结果（最后一轮结果和yolo线性图），配置参数等。
